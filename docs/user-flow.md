# User Flow

## Основной сценарий MVP

```text
/start
  -> приветствие
  -> проверка регистрации
      -> если новый пользователь:
          -> выбор родного языка
          -> выбор изучаемого языка
          -> выбор уровня
          -> сохранение профиля
      -> если пользователь существует:
          -> переход в главное меню

Главное меню
  -> Учить слова
      -> выбор набора
      -> просмотр карточек
  -> Пройти тест
      -> выбор набора
      -> прохождение квиза
      -> результат
  -> Профиль
      -> данные пользователя
      -> статистика
  -> Помощь
      -> описание возможностей
```

## Принципы UX

- у каждого экрана есть понятный следующий шаг
- нет тупиковых веток
- главное действие доступно в 1-2 нажатия
- тексты короткие и однозначные

## Что показывать на первом milestone

- команду `/start`
- схему регистрации пользователя
- логику главного меню
- разбиение будущего функционала на разделы
