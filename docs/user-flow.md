# User Flow

## Основной Сценарий MVP

```text
/start
  -> приветствие
  -> проверка регистрации
      -> если новый пользователь:
          -> выбор родного языка
          -> выбор изучаемого языка
          -> выбор уровня
          -> сохранение профиля
      -> если пользователь существует:
          -> переход в главное меню

Главное меню
  -> Изучить слова
      -> выбор темы
      -> просмотр слов и примеров
  -> Пройти квиз
      -> выбор формата квиза
      -> выбор темы
      -> прохождение квиза
      -> результат
  -> Профиль
      -> данные пользователя
  -> Статистика
      -> количество квизов
      -> точность
      -> общий прогресс
  -> Помощь
      -> описание возможностей
```

## Темы И Уровни Для MVP

Темы:

- Путешествия
- Еда и напитки
- Работа и профессии
- Семья и друзья
- Хобби и досуг

Уровни:

- A1
- A2
- B1

## Форматы Квизов

- Выбор правильного варианта
- Соответствие
- Заполнение пропусков
- Угадай слово по определению

## Сценарий 1: Изучение Слов

1. Пользователь запускает бота.
2. Бот приветствует пользователя и предлагает выбрать действие.
3. Пользователь выбирает изучение слов.
4. Бот предлагает выбрать тему.
5. Пользователь выбирает тему.
6. Бот показывает набор слов по теме.
7. Пользователь изучает слова и примеры.

## Сценарий 2: Прохождение Квиза

1. Пользователь запускает бота.
2. Бот приветствует пользователя и предлагает выбрать действие.
3. Пользователь выбирает прохождение квиза.
4. Бот предлагает выбрать формат квиза.
5. Пользователь выбирает формат и тему.
6. Бот задает вопросы.
7. Пользователь отвечает на вопросы.
8. Бот показывает итоговый результат.

## Принципы UX

- у каждого экрана есть понятный следующий шаг
- нет тупиковых веток
- главное действие доступно в 1-2 нажатия
- тексты короткие и однозначные
