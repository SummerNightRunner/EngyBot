# EngyBot

MVP Telegram-бота для изучения иностранных языков. В репозитории есть backend-приложение на `aiogram`, PostgreSQL в Docker, стартовые учебные наборы, онбординг, карточки слов, квизы и статистика.

## Что уже готово

- Python-проект под `aiogram 3`
- конфигурация через `.env`
- асинхронное подключение к PostgreSQL через `SQLAlchemy`
- контейнеризация через `Docker Compose`
- модель пользователя, учебных наборов и результатов тестов
- онбординг пользователя с сохранением профиля в БД
- стартовые учебные наборы и карточки слов
- квизы по наборам слов
- статистика по результатам обучения
- документы по архитектуре, БД и плану работ

## Стек

- Python 3.12
- aiogram 3
- SQLAlchemy 2

## Что именно мы делаем

Мы делаем backend-ориентированный Telegram-бот для микрообучения иностранным языкам. Пользователь проходит онбординг, выбирает язык и уровень, учит слова по тематическим наборам, проходит мини-тесты и отслеживает прогресс. Telegram выступает клиентским интерфейсом, вся логика и данные находятся на серверной стороне.

## Цель проекта

Разработать MVP Telegram-бота для изучения иностранных языков, который позволит пользователю:

- зарегистрироваться и настроить профиль
- изучать слова и фразы по тематическим наборам
- проходить мини-тесты для проверки знаний
- отслеживать личный прогресс и результаты обучения

## Быстрый запуск через Docker

1. Создать `.env`:

```bash
cp .env.example .env
```

2. Указать токен, полученный у `@BotFather`.

3. Запустить контейнеры:

```bash
docker compose up --build
```

4. Открыть бота в Telegram и отправить `/start`.

## Локальный запуск без Docker

1. Создать виртуальное окружение:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Установить зависимости:

```bash
pip install -e .
```

3. Создать `.env`:

```bash
cp .env.example .env
```

4. Поднять PostgreSQL локально или изменить `DATABASE_URL` на доступную БД.

5. Запустить бота:

```bash
python -m bot
```
