# EngyBot

MVP Telegram-бота для изучения иностранных языков. Репозиторий подготовлен под первый спринт кейса: здесь уже есть каркас backend-приложения, PostgreSQL в Docker, схема данных, базовые сценарии и документация для первого показа.

## Что уже готово

- Python-проект под `aiogram 3`
- конфигурация через `.env`
- асинхронное подключение к PostgreSQL через `SQLAlchemy`
- контейнеризация через `Docker Compose`
- модель пользователя и учебных наборов
- стартовый `FSM`-сценарий онбординга
- команда `/start` и главное меню
- документы для спринта 1: архитектура, БД, user flow, backlog, роли

## Стек

- Python 3.12
- aiogram 3
- SQLAlchemy 2
- PostgreSQL 16
- Docker Compose
- Long Polling

## Что именно мы делаем

Мы делаем backend-ориентированный Telegram-бот для микрообучения иностранным языкам. Пользователь проходит онбординг, выбирает язык и уровень, учит слова по тематическим наборам, проходит мини-тесты и отслеживает прогресс. Telegram выступает клиентским интерфейсом, вся логика и данные находятся на серверной стороне.

## Быстрый запуск через Docker

1. Создать `.env`:

```bash
cp .env.example .env
```

2. Указать токен, полученный у `@BotFather`.

3. Запустить контейнеры:

```bash
docker compose up --build
```

4. Открыть бота в Telegram и отправить `/start`.

## Локальный запуск без Docker

1. Создать виртуальное окружение:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Установить зависимости:

```bash
pip install -e .
```

3. Создать `.env`:

```bash
cp .env.example .env
```

4. Поднять PostgreSQL локально или изменить `DATABASE_URL` на доступную БД.

5. Запустить бота:

```bash
python -m bot
```
